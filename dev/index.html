TensorsLite.AbstractTensor — Type
AbstractTensor{T, N} <: AbstractArray{T, N}

Supertype of all Tensor types. Represents any Nth order tensor with eltype T.

TensorsLite.Tensor — Type
Tensor{T, N, Tx, Ty, Tz} <: AbstractTensor{T, N}

A Nth order tensor with eltype T. Higher order tensors are implemented as vectors of lower order tensors.
